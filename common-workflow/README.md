# Common Workflow

1. `01-analyze-qa.md`
   - 分析需求、釐清問題、與使用者確認

2. `02-plan-and-split-steps.md`
   - 切 branch、拆步驟、建立 agents-plan

3. `03-implement-code.md`
   - 依 plan-step 實作、逐步確認、review、commit

4. `04-test.md`
   - 完整測試、修 bug、重複直到 plan-step 完成

5. `05-code-review-security-and-improvements.md`
   - 深度 code review、優化、安全報告

6. `06-explain-changes-and-final-review.md`
   - 最終變更說明、總結、交付、結案確認
